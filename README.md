# web_scrap21

<h3>The model present here scrapes the website "theweatherchannel" powered by IBM,
the model doesnt require any inputs from user and acquires the data for every 4 hrs untill keyboard interrupt occurs</h3>

<ul>
<li>selenium</li>
<li>pandas</li>

</ul>
